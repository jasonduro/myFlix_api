Documentation Page

This is where all the magical documentation for the movie api app takes place!

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user /movies/:title GET None A JSON object holding data about a specific movie containing a description, genre, director, image url, and featured or not
Return data about a genre (description) by name/title (e.g. Thriller) /movies/genre/:genreName GET None A JSON object holding data about genre
Return data about a director (bio, birth year, death year) by name /movies/director/:directorName GET None A JSON object holding data about the director
Allow new users to register /users POST None A JSON object holding data about the user to add, structured like: { id:1, name: "Mikey" favorite movie: "Interstellar" }
Allow users to update their user info (username, password, email, date of birth) /users/:username PUT None A JSON object holding data about the user
Allow users to add a movie to their list of favorites /users/:username/:movieTitle POST None A text message saying movie title has been added to array
Allow users to remove a movie from their list of favorites /users/:username/:movieTitle DELETE None A text message saying movie has been removed
Allow existing users to deregister /users/:username DELETE None A text message saying user has been deleted
